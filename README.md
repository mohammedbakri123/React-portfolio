## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Running a Project

Navigate to any project folder and install dependencies:

```bash
cd "fast-react-pizza"
npm install
npm run dev
```

Each project has its own setup — check the individual project README for specific instructions.
